# dumi

## Getting Started

在线地址: [https://dumi.chen929104.vercel.app/](https://dumi.chen929104.vercel.app/)

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
